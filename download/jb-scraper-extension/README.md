# JB Shopee Scraper — Chrome Extension

Cara pakai:
1. Buka Edge/Chrome
2. Ketik di address bar: edge://extensions (atau chrome://extensions)
3. Aktifkan "Developer mode" (toggle kanan atas)
4. Klik "Load unpacked"
5. Pilih folder ini (jb-scraper-extension)
6. Icon JB muncul di toolbar!

Cara scrape:
1. Buka shopee.co.id
2. Cari produk yang mau ditambah ke JB
3. Klik icon JB Scraper di toolbar
4. Pilih kategori, klik "Ambil Semua Produk di Halaman Ini"
5. Buka halaman lain, klik lagi (data nyimpel otomatis)
6. Kalau udah cukup, klik "Copy CSV"
7. Buka JB admin → Bulk Upload → paste → upload!
