🎯 Scrape Produk (Halaman Aktif)

Buka halaman produk → klik tombol. Image Tokopedia auto-upload ke Cloudinary, AT link auto-generate.

🔗 Paste Link Produk

Paste URL produk (1 per baris). Auto scrape + mirror image + AT link.

🔄 Refresh Image Tokped

Re-fetch image Tokopedia yang expired → upload ke Cloudinary (permanent). Scraper buka tab background, download image, upload ke JB. Butuh Admin Secret di Config.

❓ Cara Pakai
  1. Isi Admin Secret (password admin JB Anda)
  2. Buka produk di marketplace mana saja (Shopee/Tokopedia/TikTok/Blibli/Lazada/Bukalapak/Zalora/Sociolla)
  3. Klik "Tambah ke Koleksi"
  4. Ulangi untuk produk lain (bisa 10, 20, 30 produk)
  5. Klik "Upload Semua" → semua masuk DB sekaligus
  6. Klik "Clear" untuk reset koleksi

Note: Marketplace auto-detect dari URL. Affiliate URL Shopee auto (AT), lainnya manual di admin.
